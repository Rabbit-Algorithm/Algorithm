## 🧠 스터디 방식

- **첫째주 ~ 셋째주 (또는 넷째주)**  
  → 2주 주기로 알고리즘 유형 파악 및 풀이

- **마지막 주 (주말 오프라인)**  
  → 한 달 동안 풀었던 문제 중 **랜덤 1문제 선택**,  
  → 각자 **풀이 설명 및 코드 리뷰**
